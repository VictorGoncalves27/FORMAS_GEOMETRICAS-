Calculadora de Area do Trapezoide
O trapezoide irregular é um polígono de quatro lados no qual dois deles são paralelos, mas os outros dois não o são. Além disso, os ângulos entre esses lados não paralelos podem variar, assim como os comprimentos dos quatro lados do trapézio. Para determinar a área de um trapezoide não convencional, normalmente é preciso empregar técnicas geométricas mais complexas ou estimativas, de acordo com as particularidades do trapezoide em análise.
Formula para calcular : Área = (base maior + base menor) * altura / 2
